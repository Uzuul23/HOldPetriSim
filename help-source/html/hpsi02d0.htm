Export Import

Besides the binary file format with the extension hps provides HPSim also a file format with the extension hpx for the purpose of compatibility with other versions of HPSim. In other words, a file saved in this format will be also accessible by future versions of HPSim. Be notified that editing such files with a text editor can corrupt the information and make them nonreversible useless to HPSim.

To export a file in the hpx format you have to select the menu entry “FileàExportàFile” and choose a file name in the file save dialog then.

The respective import function for the hpx file type is selected by the menu entry “FileàImportàFile” and by choosing the desired file in the file open dialog then.

HPSim provides the feature to save the net as Windows Bitmap. To save the entire net as bitmap, select the menu entry “FileàExportàDocument”. To save only the currently visible part of the net as bitmap, select the menu entry “FileàExportàView.

The menu entry “FileàExportàNet” provides the possibility to save the currrent net as a text file. This format enables you to import the file in other application for further analysis of the net.

Following features of the net will be exported:

  • Naming of Transitions and Places

  • Inzidenzmatrix

  • Marking Vector

  • Arc Types
