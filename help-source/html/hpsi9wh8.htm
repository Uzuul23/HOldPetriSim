Label

The label holds a special meaning among the passive graphical objects.

The user can not delete or add labels.

Labels are always automatically attached when a graphical object (further referenced as owner of the label) is created, and represents a property of it.

The position of the label to the attached graphical object can be changed by dragging it with the mouse. When the graphical object is moved, the label is automatically moved along.

The value of the label is only changed, when the property of its attached graphical object (owner) has changed.

The position of the labels in relation to its owner’s position can be reset to its original position by choosing the menu entry ”Reset Labels” from the context menu.
