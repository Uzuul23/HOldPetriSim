Rectangle

Select the tool “Rect” from the “Tools” menu.

The object rectangle can be used to represent a rectangle with rounded corners and sharp ones, and an ellipsis (this can be formed as a circle when desired).

The object can be placed and scaled without restrictions.

To adjust the radius of the corners of a ”RoundRec”, you have to select the context menu entry ”SetRound”. Additional mouse handles on the object will now be displayed. Drag one of those additional handles to adjust the radius to your needs.
