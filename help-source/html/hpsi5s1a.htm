Transition

Select the tool “Transition” from the “Tools” menu.

This active graphical object represents a transition in a Petri Net. The enabling time for a transition can be defined. After the enabling time has elapsed, the transition will be switched instantaneously. Various functions are provided to represent the enabling time behavior of the transition. It can be chosen among: Immediate, Deterministic, Exponential Distribution and Uniform Distribution.

It has to be taken into account that the required random numbers are provided by a random number generator.

The time mode Deterministic uses the property Initial Delay as defined enabling time.

The time mode Exponential Distribution uses the property Initial Delay as mean value for the distribution function of the enabling time.

The time mode Uniform Distribution uses the property Initial Delay as lower bound and the property Range Delay as upper bound (in absolute values) for the distribution function of the enabling time.
