Context Menu

By clicking with the right mouse button, you invoke a context menu. Depending on the position of the mouse and the current mode, this context menu can consist of different menu entries.

When the context menu is invoked over the main frame, the context menu entries are identical to the menu “View” of the main menu.

When the context menu is invoked over an Editor Window, the context menu entries are identical to the menu “Arrange” of the main menu. More menu entries of the context menu will be provided, when the menu was invoked over an object.
