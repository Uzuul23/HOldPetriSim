Simulation Run Fast

The Run Fast simulation mode will be invoked, when selecting the menu entry “SimulationàRun Fast”.

The calculation of the net is performed with the highest possible speed.

The updated token will not be displayed in the view after each calculation of the net for one step. An synchronization of the calculated token position with the displayed net occurs only after the simulation has been stopped.

The simulation is stopped when the conditions are met as described in the Simulation Run chapter of this help document.
