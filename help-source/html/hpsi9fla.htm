Step 2: Drawing the Net

Hint: When trying to select an object for the reason of repositioning, resizing or changing its properties through the ”Project Explorer”, you need first to unselect the current tool by choosing the menu entry ”ToolsàSelect”.

  • Select the tool “Place” from the tools menu and place them with the mouse cursor in the document as shown in the picture below.

  • Select the tool “Transition” from the tools menu and place them with the mouse cursor in the document as shown in the picture below.

  • Select the tool “Arc” from the tools menu to connect the places and transition with each other in the document as shown in the picture below. The connection can be drawn by dragging the mouse from source to sink object (which has to be a transition and a place or vice versa)

  • The path of the arcs can be rearranged by dragging them on their mouse handles (small visible squares when the arc is selected)

  • Save now your work by selecting the menu entry “FileàSave”. HPSim does not provide a undo-functionality.
