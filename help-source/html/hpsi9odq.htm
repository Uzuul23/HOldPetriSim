Zooming

The zoom functionality of HPSim provides a convenient way to enlarge or reduce the view of documents.

When selecting the menu entry “ZoomàZoom In”, you can gradually enlarge the current view by clicking with the mouse in the window. Alternatively, you can drag a square with the mouse to zoom in a desired region.

When selecting the menu entry “ZoomàZoom Out”, you can gradually reduce the current view by clicking with the mouse in the window.

When selecting the menu entry “ZoomàZoom 1:1”, the current view will be set to ist real size.

When selecting the menu entry “ZoomàZoom Port”, the current view will be set so that the entire document - as specified in the ”ExtraàPropertiesàEditoràView” - fits in the window under consideration of the proportion of the document.
