Step 5: Simulation

  • No we like to conduct a Token Game Animation with the created net.

  • For that, we switch to simulation mode of HPSim by selecting the menu entry “SimulationàSim Mode”. The Status Window will be opened automatically to show us important status messages during simulation.

  • You can either close the Status Window or force it to the background of the main frame by clicking on the window controls in the upper right corner to see the animated tokens of your net during simulation.

  • We start now the actual simulation by selecting the menu entry “SimulationàRun Normal”. When done everything according to the instructions given, you should see now the tokens moving between the different places.
