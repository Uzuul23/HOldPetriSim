Project Explorer

The Project Explorer provides an interface to conveniently view and edit the properties of an object. When the simulation is in online mode, only the properties of active objects can be viewed or edited. The window of the Project Explorer can either be docked to any side of the main frame or free floating by dragging it with the mouse.

The Project Explorer window can activated or deactivated by selecting the menu entry ”ViewàProject Explorer”.

If the Project Explorer is not docked to the main frame and deactivated, then a selection of an object will automatically open the Project Explorer window. When deselecting the object, the Explorer Window will be closed again. This behavior of the Project Explorer window can be deactivated in the checkbox ”EditoràParanoiaàPopup Project Explorer” under the menu entry ”ExtraàProperty”
