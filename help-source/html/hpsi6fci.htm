Splitting a Window

By using the window divider (one divider above the vertical scroll bar and one divider left to the horizontal scroll bar of each Editor Window) you can have up to four views of the same net in one window displayed. This features enables you to work on up to four different regions of the net with each having its individual zoom. Just drag the divider with the mouse to the desired position to divide an Editor Window. The picture displays an Editor Window showing four different regions of the document.
