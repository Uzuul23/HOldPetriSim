Status Bar

The status bar provides some useful information.

In the field ”Current Information” you will be informed about hints on the currently used function, for time consuming functions a progress bar will be displayed here.

The field ”Status Simulator” displays Online when the simulator is in online mode, or else Offline is displayed.

The field ”Step Count” displays current number of steps of a simulation.

The field ”Time” displays the elapsed real time for the current simulation in millisecond.

The field ”Performance” displays first the idling time between two consecutive calculation steps in millisecond, and second the time used to calculate one step in millisecond.

The field ”Position” displays the x,y-coordinates of the current cursor position in the current window in pixels. The origin of the coordinates is the upper left corner of the window.
