Change the Properties of an Object

Some of the properties of an object can be change through the context menu. To change the object properties through the context menu, you just have to place the mouse pointer over that object and click the right mouse button to invoke the context menu. All properties of an object can be viewed and changed in the Project Explorer window. In the left column of that window, you can find the name of the property, and in the right column its value. To change a value, click in the cell with value. Depending on the kind of information you want to change, a direct input, a drop down list or a dialog is provided.
