Document Properties

To change the properties of the current document, select the menu entry “ExtraàProperties”. The settings are only valid for the current document, and will be saved and loaded along with the document.

Editor

  • Size of View: Here you can assign the number of pixels for the horizontal and vertical extension of the view. Possible values are from 100 up to 15000 pixels.

  • Background Color: Here you can choose the desired background color for the view.

  • Message Window Here you can select, whether you want the Message Window be started automatically each time you enter the simulation mode.

  • Project Explorer: Here you can select, whether you want the Project Explorer be started automatically each time you select an object.

Grid

  • Grid: Here you can select the visibility of the grid and the distance in pixels.

  • Net Objects: When selected, net objects will be aligned to grid.

  • Draw Objects: When selected, draw objects will be aligned to grid.

Simulation

  • Sample Time: Here you can select the time base of the simulation (distance between two consecutive simulation steps)

  • Stop Running: Here you can select when the simulation should stop automatically by either reaching a certain simulation time or step counter.

  • Output File: Here you can choose an output file for the simulation data.

Defaults

This feature is not available in this version.
