Printing

HPSim provides the feature to print the current view with any windows printer.

The printer dialog box will be invoked, when selecting the menu entry “FileàPrint”.

The page to be printed can be previewed, when selecting the menu entry “FileàPane View”.

The view - as its dimension were specified in the Editor register of the property dialog - will be proportionally (proportion of horizontal to vertical will be preserved by HPSim) adjusted to fit on page. To optimize the consumption of the space for the drawing on the page, choose a view format in the property box which is close to the used page format.
