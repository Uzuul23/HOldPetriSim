Starting a Simulation

The mode can be toggled between Simulator and Editor, when selecting the menu entry “SimulationàSim Mode”. When entering Simulator mode, you will be prompted to save your work if necessary. As well as the Status Window will automatically popup.

Following commands are exclusively in the Simulator mode available: Step, Run, Run Fast.

When being in Simulator mode, only the properties of active graphical objects, like transitions, arcs and places, can be viewed or edited. Scaling or displacement of objects is not possible at all now.

The grid marker are generally switched off in Simulator mode.
