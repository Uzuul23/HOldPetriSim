Step 1: Document Properties

  • Select the menu entry “ExtrasàProperties”

  • We choose a background color for the working space as shown below.

  • The logical size of the document will be set to (length x height) 1000 x 1500 pixel

  • We choose the register “Grid” to adjust the setting for the grid as shown below.

  • Keep the standard settings as follows: Grid: Visible, Grid Spacing: 10 points, Align to Grid: Net Objects
