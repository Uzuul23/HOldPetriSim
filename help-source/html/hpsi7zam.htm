Load and Save

To load an existing document you can choose among the following ways:

  • Select menu entry “File Open” and select the file in the file open box

  • By “drag&drop” drop the hps-file into the open HPSim window

  • By double-clicking on a hps-file

To save a document you can select the menu entry ”File Save”. By selecting the menu entry ”File SaveAs” you can save the current document under a new name.
