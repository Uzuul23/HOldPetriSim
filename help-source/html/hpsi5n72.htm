Tool Bars

The main frame of HPSim contains a number of tool bars, which can be added/removed and positioned to the user’s needs through the menu. The tool bars of HPSim can be seen in the picture. The iconified function will be selected by clicking with the mouse on the button. Functions, which are currently not available, will be gray shaded.

When selecting the menu entries in the menu “View”, the various tool bars can be switched on/off as desired.
