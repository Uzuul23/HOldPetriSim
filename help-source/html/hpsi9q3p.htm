Step 3: Adjusting the Net Object Properties

  • The properties of a selected object can be edited with the “Project Explorer” as shown below. Bear in mind that you have first unselect the current tool to ”ToolsàSelect” to do so.

  • We change now the name of the object P0 in “Storage”, the capacity to 1000 and the number of initial tokens to 3 by first selecting the object with the mouse cursor, and then changing the value of the property in the “Property Explorer” window..

  • We change the weight of the arc leading from that place P0 to 2 by first selecting the object with the mouse, and then changing the value of the property in the “Property Explorer” window.

  • We change the value of the inital tokens for object P2 to 1 by first selecting the object with the mouse, and then changing the value of the property in the “Property Explorer” window.
