Polygon

Select the tool “Polygon” from the “Tools” menu. .

The object polygon can be used to represent a polyline – a linked line which connects several points. When the first point and the last connected points are identical, the line represents a polygon.

The polygon can either be filled with a color or transparent.

The position of the entire polygon at once or only for single points can be changed.

New points can be added by clicking between two points of the line.

One or all points can be deleted by selecting the menu entry ”Delete Point” or ”Del All Points” of the context menu, when the mouse is over the object.
