Activation of the Output File

You can cause HPSim to write an output file with the relevant simulation data (Markierungsvektor) in the common csv-file format (this format is understood by many applications, such as Excel), when selecting the menu entry “SimulationàWrite Output File”.

The Markierungsvektor is written for each simulation step to that file.

The path to that file can set in ”ExtraàProperties” dialog box.

Be aware that this can quickly reach a reasonable size by long simulations with big nets.

You can further analyze the simulation data by a spread sheet application like MS-Excel™
