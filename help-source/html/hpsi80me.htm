Simulation Run

The Run simulation mode will be invoked, when selecting the menu entry “SimulationàRun”. The updated token position can be seen, after each calculation of the net for one step.

The continuous simulation is stopped when one of the following condition is met:

  • The limit of the simulation time is reached OR

  • The limit of the simulation steps is reached OR

  • A ”Dead Lock” has been detected OR

  • The simulation is interrupted by the user

The limit for simulation time and steps can be set in the “ExtraàPropertiesàSimulation” dialog box.

The delay time between two consecutive simulation steps can be gradually adjusted by the menu entries “SimulationàIncrease Speed” and “SimulationàDecrease Speed”.

The delay time between two consecutive simulation steps can be infinitely variable adjusted with the slider of the tool bar “Symbar”.
