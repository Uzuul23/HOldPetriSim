Arc

Select the tool “Arc” from the “Tools” menu.

This active graphical object represents an arc in a Petri Net. The arc connects transition with places in a specified direction. A connection can only be established, when it is according to the semantic of a Petri Net.

An arc is created by dragging the mouse from a source object (can be either a transition or a place) to a sink object (when the source object was a transition then the sink object has to be a place, and vice versa). When HPSim fails to create an arc as specified, a descriptive dialog box will appear. The created arc can later be formatted like a polyline by dragging its mouse handlers.

HPSim provides not only the standard arc, it also provides test arcs and inhibitor arcs.
